发表于 2018-03-24 | 分类于 blog | | 阅读次数:
原本在Win下的博客由于使用环境太久没更新,修复过程出了各种bug,索性重新装了一次,顺便备份恢复到了现在常用的linux环境下(说得好像我会经常更新一样).在此简单说一下恢复的过程

环境:

os: Debian 9.3 stretch
git: 2.11.0
node: 8.9.3
npm: 5.8.0
hexo-cli:1.1.0

1. 关联github

1.1 设置git

设置git私钥

安装openssh,生成SSHkey

确保ssh-agent可运行

添加SSH key

拷贝 ~/.ssh/id_rsa.pub 中的内容，然后，在 GitHub 右上方点击头像，选择 ”Settings”，在右边的 ”Personal settings” 侧边栏选择 ”SSH Keys”。点击 ”Add key” 按钮添加，接着粘贴 key。
